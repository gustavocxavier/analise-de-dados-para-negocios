Módulo 1: Fundamentos do Google Colab e Python#

Bem-vindos ao início da nossa jornada de análise de dados! Neste módulo vamos entender o ambiente do Google Colab, aprender o básico de Python e conhecer as bibliotecas que vão transformar a forma como você trabalha com dados.

1.1 O Ambiente Google Colab#

  • Computação em Nuvem: O Colab executa seu código nos servidores do Google.

  • Sem Instalação: Você só precisa do seu navegador.

  • Células: O Notebook é dividido em células de Texto e células de Código.

Célula de Código#

Ao clicar no botão de ‘play’ ao lado de uma célula de código, ela é executada. O resultado aparece logo abaixo. Experimente executar a célula a seguir! Lembre-se que esse é o primeiro código que você roda, então ele demora alguns segundos para começar a rodar.

print('Olá, Negócios!')

Olá, Negócios!

1.2 Lógica Básica de Python#

Aqui usaremos variáveis para armazenar informações de negócio. Pense em variáveis como caixinhas com etiquetas: você guarda um valor dentro (como receita = 15000) e depois pode usá-lo em qualquer cálculo.

receita = 15000
custo = 8000
lucro = receita - custo
print(f'O lucro foi de R${lucro}') # Exibe o lucro formatado como moeda

O lucro foi de R$7000

1.3: O Poder das Bibliotecas#

Imagine que o Python puro é um celular novo que acabou de sair da caixa. Ele faz ligações e manda mensagens, mas para fazer coisas incríveis, você precisa instalar aplicativos.

No Python, esses aplicativos são chamados de Bibliotecas (ou Libraries). Para que o Python possa usá-las, precisamos importá-las no início do nosso código. Vamos usar as mais famosas no mundo dos negócios:

  • Pandas: Para ler e organizar dados (como o Excel).

  • Seaborn: Para criar gráficos bonitos e profissionais com facilidade.

  • Matplotlib: A biblioteca gráfica base do Python, usada junto com o Seaborn para personalização extra.

  • Pingouin: Para testes estatísticos de hipóteses, com resultados em formato de tabela legível.

# Importando nossas bibliotecas (nossos "aplicativos")
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Configurando um tema visual bonito para os gráficos
sns.set_theme(style="whitegrid")

1.3.1. Séries Temporais (Gráfico de Linha)#

Vamos analisar um conjunto de dados real sobre a quantidade de passageiros em voos ao longo dos anos. Veja como é simples criar um gráfico de série temporal com apenas uma linha de código principal.

# Carregando os dados de voos que já vem embutidos no Seaborn
voos = sns.load_dataset("flights")

# Criando o gráfico de linha (Série Temporal) com xlabel e ylabel
sns.lineplot(data=voos, x="year", y="passengers")
plt.xlabel("Ano")
plt.ylabel("Passageiros")

Text(0, 0.5, 'Passageiros')

1.3.2. Analisando Categorias (Boxplot)#

Agora, vamos imaginar que você gerencia um restaurante. Você quer saber: Em qual dia da semana os clientes gastam mais dinheiro?

O Boxplot é o gráfico perfeito para relacionar uma categoria (Dia da Semana) com um número contínuo (Valor da Conta). Ele mostra a mediana, os quartis e os valores atípicos (outliers) de forma compacta.

# Carregando dados de gorjetas e contas de um restaurante
gorjetas = sns.load_dataset("tips")

# Criando o Boxplot
sns.boxplot(data=gorjetas, x="day", y="total_bill", hue="day", palette="Set2", legend=False)

<Axes: xlabel='day', ylabel='total_bill'>

As bibliotecas vistas são as mais importantes para análise de dados, mas existem muitas outras que podem ser úteis dependendo do tipo de análise que você deseja fazer. O importante é saber que o Python tem uma vasta gama de ferramentas à sua disposição para transformar dados brutos em insights valiosos para o seu negócio.

Outras bibliotecas populares incluem:

  • NumPy: Para cálculos numéricos avançados.

  • Matplotlib: Para gráficos personalizados e detalhados.

  • Plotly: Para gráficos interativos e visualizações avançadas.

  • Scikit-learn: Para machine learning e análise preditiva.

Mas para este curso, focaremos principalmente em Pandas e Seaborn, pois são essenciais para a maioria das análises de dados que precisamos fazer.

📊 Exemplo de Negócio — Módulo 1#

Cenário: Você é analista em uma distribuidora. O gestor quer saber rapidamente o lucro líquido de uma venda específica e pede que você crie um mini-script em Python para calcular e exibir o resultado.

Execute a célula abaixo e observe como o Python funciona como uma calculadora inteligente.

# Exemplo de Negócio – Módulo 1
# Dados de uma venda da distribuidora
preco_venda   = 4500.00  # Receita da venda
custo_produto = 2800.00  # Custo do produto
frete         = 150.00   # Custo de frete
impostos      = 360.00   # Impostos sobre a venda

lucro_liquido = preco_venda - custo_produto - frete - impostos

print(f'Receita:          R$ {preco_venda:,.2f}')
print(f'Custos totais:    R$ {(custo_produto+frete+impostos):,.2f}')
print(f'Lucro líquido:    R$ {lucro_liquido:,.2f}')
print(f'Margem de lucro:  {lucro_liquido/preco_venda*100:.1f}%')

Interpretação: A célula acima mostra a margem de lucro daquela venda. Se a margem cair abaixo de 10%, pode ser necessário renegociar o custo do produto ou o frete.
