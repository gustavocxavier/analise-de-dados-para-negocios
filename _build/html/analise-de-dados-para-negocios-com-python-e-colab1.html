Módulo 1: Fundamentos do Google Colab e Python#

Bem-vindos ao início da nossa jornada de análise de dados! Neste módulo vamos entender o ambiente do Google Colab, aprender o básico de Python e conhecer as bibliotecas que vão transformar a forma como você trabalha com dados.

1.1 O Ambiente Google Colab#

  • Computação em Nuvem: O Colab executa seu código nos servidores do Google.

  • Sem Instalação: Você só precisa do seu navegador.

  • Células: O Notebook é dividido em células de Texto e células de Código.

1.3: O Poder das Bibliotecas#

Imagine que o Python puro é um celular novo que acabou de sair da caixa. Ele faz ligações e manda mensagens, mas para fazer coisas incríveis, você precisa instalar aplicativos.

No Python, esses aplicativos são chamados de Bibliotecas (ou Libraries). Para que o Python possa usá-las, precisamos importá-las no início do nosso código. Vamos usar as mais famosas no mundo dos negócios:

  • Pandas: Para ler e organizar dados (como o Excel).

  • Seaborn: Para criar gráficos bonitos e profissionais com facilidade.

  • Matplotlib: A biblioteca gráfica base do Python, usada junto com o Seaborn para personalização extra.

  • Pingouin: Para testes estatísticos de hipóteses, com resultados em formato de tabela legível.

1.3.2. Analisando Categorias (Boxplot)#

Agora, vamos imaginar que você gerencia um restaurante. Você quer saber: Em qual dia da semana os clientes gastam mais dinheiro?

📊 Exemplo de Negócio — Módulo 1#

Cenário: Você é analista em uma distribuidora. O gestor quer saber rapidamente o lucro líquido de uma venda específica e pede que você crie um mini-script em Python para calcular e exibir o resultado.

Execute a célula abaixo e observe como o Python funciona como uma calculadora inteligente.

# Exemplo de Negócio – Módulo 1
# Dados de uma venda da distribuidora
preco_venda   = 4500.00  # Receita da venda
custo_produto = 2800.00  # Custo do produto
frete         = 150.00   # Custo de frete
impostos      = 360.00   # Impostos sobre a venda

lucro_liquido = preco_venda - custo_produto - frete - impostos

print(f'Receita:          R$ {preco_venda:,.2f}')
print(f'Custos totais:    R$ {(custo_produto+frete+impostos):,.2f}')
print(f'Lucro líquido:    R$ {lucro_liquido:,.2f}')
print(f'Margem de lucro:  {lucro_liquido/preco_venda*100:.1f}%')

Interpretação: A célula acima mostra a margem de lucro daquela venda. Se a margem cair abaixo de 10%, pode ser necessário renegociar o custo do produto ou o frete.

2.2 Inspeção e Limpeza#

Dados reais são bagunçados! Eles vêm com informações faltando (valores nulos ou NaN) e linhas duplicadas. Se não limparmos isso, nossas análises e decisões de negócios estarão erradas.

Vamos inspecionar a saúde da nossa base.

# LIMPEZA DE DADOS

Bônus: O Mundo “Clássico” do Python (Matplotlib)#

Você deve estar se perguntando: “Se o Seaborn é tão fácil, por que as pessoas usam outras coisas?”

A verdade é que o Seaborn é construído em cima de uma biblioteca muito mais antiga e famosa chamada Matplotlib. O Matplotlib é extremamente poderoso, mas ele é “manual”.

Para provar como o Seaborn facilita a sua vida nos negócios, veja a diferença para criar um simples gráfico de barras com a média de vendas por região:

# O JEITO SEABORN (Automático)
# Ele mesmo já junta as regiões iguais e calcula a média das vendas sozinho!
sns.barplot(data=df_vendas, x='Regiao', y='Faturamento', color='blue')

# O JEITO MATPLOTLIB (Manual)
import matplotlib.pyplot as plt

# 1. Primeiro precisamos agrupar os dados e calcular a média manualmente (tipo Tabela Dinâmica)
media_vendas = df_vendas.groupby('Regiao')['Faturamento'].mean()

# 2. Depois temos que passar os eixos separadamente para desenhar o gráfico
plt.bar(media_vendas.index, media_vendas.values, color='gray')
plt.show()

Resumo para Negócios: Use o Matplotlib quando precisar desenhar do zero ou fazer ajustes muito finos. Use o Seaborn para transformar suas tabelas em insights visuais estatísticos em segundos!

📊 Exemplo de Negócio — Módulo 3#

Cenário: O time de marketing quer uma visualização rápida para apresentar na reunião de resultados. Você precisa mostrar a relação entre investimento em marketing e vendas geradas, com um gráfico de dispersão profissional.

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
sns.set_theme(style='whitegrid')

df_mkt = pd.read_csv('https://raw.githubusercontent.com/gustavocxavier/analise-de-dados-para-negocios/refs/heads/main/Dados/marketing.csv')

# Gráfico de dispersão com linha de tendência
sns.regplot(data=df_mkt, x='Investimento', y='Vendas',
            scatter_kws={'alpha':0.5}, line_kws={'color':'red'})

plt.title('Investimento em Marketing vs. Vendas Geradas', fontsize=14)
plt.xlabel('Investimento (R$)')
plt.ylabel('Vendas (R$)')
plt.tight_layout()
plt.show()

Interpretação: A linha vermelha é a linha de tendência (regressão linear). Se os pontos acompanham bem a linha, o investimento em marketing tem um impacto consistente nas vendas. Pontos muito distantes da linha são campanhas com resultado anormal (muito acima ou abaixo do esperado).

Módulo 4: Testes de Hipóteses com Pingouin#

Em negócios, nós não gostamos de adivinhar. Se a nova página do site (Versão B) vendeu mais que a antiga (Versão A), foi porque ela é realmente melhor ou foi apenas sorte daquela semana?

A estatística responde isso. Vamos usar a biblioteca Pingouin para fazer testes rigorosos de forma extremamente simples.

Nota: O Pingouin não vem instalado por padrão no Google Colab. Vamos instalá-lo com o comando abaixo (o ! indica que é um comando do sistema operacional, não Python).

# Instalando o Pingouin (execute esta célula apenas uma vez por sessão do Colab)
!pip install pingouin
import pingouin as pg

Tópico 4.1: Teste A/B (O Famoso Teste T)#

Cenário: Lançamos duas versões de um anúncio (A e B). Queremos saber se a diferença de cliques entre eles é real.

Regra de Ouro da Estatística para Negócios: Olhe para a coluna p-val (Valor-P) no resultado.

  • Se for MENOR que 0.05: A diferença é real e comprovada!

  • Se for MAIOR que 0.05: A diferença não é confiável (pode ter sido sorte).

# Lendo os dados do nosso Teste A/B
df_ab = pd.read_csv('https://raw.githubusercontent.com/gustavocxavier/analise-de-dados-para-negocios/refs/heads/main/Dados/teste_ab_vendas.csv')

# Separando quem viu o Anúncio A e quem viu o Anúncio B
vendas_A = df_ab[df_ab['Anuncio'] == 'A']['Vendas']
vendas_B = df_ab[df_ab['Anuncio'] == 'B']['Vendas']

# Executando o Teste T com Pingouin
pg.ttest(x=vendas_A, y=vendas_B)

Tópico 4.2: Comparando Várias Opções (ANOVA)#

O Teste T só compara duas coisas. E se quisermos testar 3 designs diferentes de embalagem nas prateleiras e ver qual vende mais?

Para comparar 3 ou mais grupos, usamos a ANOVA (Análise de Variância). O Pingouin facilita muito porque podemos passar a tabela inteira para ele.

# Lendo dados de vendas de 3 designs diferentes (Vermelho, Azul, Verde)
df_embalagens = pd.read_csv('https://raw.githubusercontent.com/gustavocxavier/analise-de-dados-para-negocios/refs/heads/main/Dados/embalagens.csv')

# Executando a ANOVA:
# data = nossa tabela
# dv = variável dependente (o que queremos medir? 'Vendas')
# between = qual coluna divide os grupos? ('Design_Embalagem')

pg.anova(data=df_embalagens, dv='Vendas', between='Design_Embalagem')

Tópico 4.3: Correlação (Causa e Efeito?)#

Queremos provar matematicamente se dar descontos maiores realmente aumenta o volume de compras.

A Correlação vai de -1 a 1. Quanto mais perto de 1, mais forte é a relação positiva (um sobe, o outro sobe).

# Lendo os dados de promoções
df_promocoes = pd.read_csv('descontos.csv')

# Testando a correlação entre a porcentagem de desconto e as unidades vendidas
pg.corr(x=df_promocoes['Desconto_Oferecido'], y=df_promocoes['Unidades_Vendidas'])

Bônus: O Mundo “Clássico” do Python (SciPy)#

Por que escolhemos o Pingouin para este curso?

No mercado de trabalho ou lendo tutoriais no Google, você fatalmente esbarrará no pacote scipy.stats. Ele é a ferramenta mais antiga e clássica para estatística em Python.

Vamos rodar o mesmo Teste A/B do início usando o scipy apenas para você ver a diferença. Note como a resposta dele é feita para matemáticos, enquanto o Pingouin gera uma tabela bonita e fácil de ler (DataFrame).

# Importando a biblioteca clássica
from scipy import stats

# Rodando o mesmo Teste T (Teste A/B) do Tópico 4.1
resultado_scipy = stats.ttest_ind(vendas_A, vendas_B)

# Veja como a apresentação do resultado é mais "crua"
print(resultado_scipy)

📊 Exemplo de Negócio — Módulo 4#

Cenário: A equipe comercial quer saber se oferecer descontos maiores realmente aumenta o volume de unidades vendidas. Vamos usar o arquivo descontos.csv e calcular a correlação com o Pingouin.

import pandas as pd
import pingouin as pg
import seaborn as sns, matplotlib.pyplot as plt
sns.set_theme(style='whitegrid')

df_desc = pd.read_csv('https://raw.githubusercontent.com/gustavocxavier/analise-de-dados-para-negocios/refs/heads/main/Dados/descontos.csv')

print('Primeiras linhas:')
print(df_desc.head())
print()

# Correlação de Pearson
resultado = pg.corr(x=df_desc['Desconto_Oferecido'], y=df_desc['Unidades_Vendidas'])
print('--- Resultado da Correlação ---')
print(resultado[['n','r','CI95%','p-val']])

Como interpretar: Observe a coluna r (coeficiente de correlação):

  • r próximo de 1: forte relação positiva — descontos maiores → mais unidades vendidas.

  • p-val < 0.05: o resultado é estatisticamente significativo, não é sorte.

Se r > 0.7 e p-val < 0.05, a política de descontos tem impacto real nas vendas!
